# API 명세

## 회원가입 API

- 엔드포인트: POST /api/auth/signup
- 요청: { email, password }
- 응답: { success, userId } 또는 { error }

## 인증

- JWT 기반 인증 토큰 발급

## 에러 코드 예시

- 400: 입력값 오류
- 409: 중복 이메일
- 500: 서버 오류
